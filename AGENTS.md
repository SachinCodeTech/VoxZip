# Project Workflow

This document outlines the standard development and backup workflow for this project.

## Development Environment
- **Platform**: [Google AI Studio](https://ai.studio/build)
- **Primary IDE**: Built-in coding agent and editor in Google AI Studio.
- **Workflow**: All major feature development, UI iterations, and logic implementations are performed directly within the AI Studio environment.

## Backup & Version Control
- **Platform**: [GitHub](https://github.com)
- **Process**: 
  - Periodic backups are performed by exporting the project code from Google AI Studio.
  - Users can use the **"Export to GitHub"** or **"Download ZIP"** options in the AI Studio settings menu to ensure code persistence outside of the platform.

## Instructions for AI Agent
- Respect the futuristic, high-utility aesthetic established in the `App.tsx` file.
- Maintain the "local-first" privacy promise (all processing happens in the browser).
- When implementing new features, ensure they align with the "VoxZip Engine" branding.
